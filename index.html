# Cipolla Protocol — Layer Zero > Experimental onchain economic mechanism for tokenized assets. > First public implementation on Base — limited ERC721 collection. --- ## Overview Cipolla Layer Zero is a Genesis NFT collection of 250 tokens deployed on Base L2, built around an experimental economic mechanism called **Provenance Royalties**. The core innovation: on every secondary resale, royalties are automatically distributed not only to the creator, but to **all historical owners** of the token — proportionally and perpetually, even after selling. The declared sale price (`salePrice`) is supplied by the caller and is **not cryptographically verified** on-chain. The contract checks coherence between the declared price and the ETH sent (msg.value should equal salePrice × 15%), but cannot verify the actual transaction price for P2P sales. For marketplace sales, the price is publicly verifiable via marketplace APIs. **Future versions** of the Cipolla Protocol will integrate price verification and automatic royalty distribution within a single atomic transaction on the Cipolla marketplace — eliminating the need for manual calls and ensuring trust-minimized distribution.
